# Framework Shield-Force
- Para desenvolvimento PHP!


### Setup (1) de instalação do Framework!

```
$: sfrun --start-system
```

### Comando de Ajuda

```
$: sfrun --help
```

### Versão Framework

```
$: sfrun --v
```

### Servidor Embutido

```
$: sfrun --start-server --port 3000 --host localhost --path /public
```
