# Framework Shield-Force
- Para desenvolvimento PHP!


### Setup de instalação do Framework!

```
$: sfrun --start-system --host 'localhost' --port '80' --path-public '/public_html'
```

### Comando de Ajuda

```
$: sfrun --help
```

### Versão Framework

```
$: sfrun --v
```

### Servidor Embutido

```
$: sfrun --start-server --host 'localhost' --port '3000' --path '/public'
```
